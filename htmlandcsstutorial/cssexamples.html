CSS Units

  1. Pixels - Broadly speaking, it's one of the dots/lights on your screen.
    • Pixel font size example - 30px
    • Pixel font size example - 10px
  2. Percentage - The size of them is relative to the size of the parent element
    • Percentage font size example - 50%(relative to the parent element font size: 50px)
    • Percentage font size example - 25%(relative to the parent element font size: 50px)
    • Percentage font size example - 40%
  3. EM/REM(relative values) - The size of them is also relative to the browser's font size
    • EM/REM font example - 1em
    • EM/REM font example - 1rem
    • EM and REM are almost the same thing - the difference is that REM will only be relative to the browser font size but EM is also relative to it's parent element.
    • Observe that if you change the zoom of your browser, the em text will become bigger, but the other fonts will stay the same size.
  4. VH and VW(view height and view width) - The size of the font will be relative to the size of your screen(not browser font)
    • 10vw, 10vh - means you are getting 10% of the screen width and height
    • 20vw, 20vh - this shape is occupying 20% of the screen's width and height
    • This will change depending on the size of your screen's dimensions(you can try it out by resizing your window)
